Degradado linear II

Ademas de la unidad deg para controlar la rotación de nuestro degradado, CSS nos permite utilizar otras unidades de medida además de palabras clave.


div {
    background-image: linear-gradient(to bottom right, HotPink, SkyBlue)
}

También podemos variar el punto en que los colores comienzan a fundirse especificando un color stop con un valor (normalmente un porcentaje) detrás del color que queramos afectar.


div {
    background-image: linear-gradient(to top, HotPink 80%, SkyBlue);
}

Si hacemos coincidir los cambios de color podemos simular transiciones entre dos colores planos.


div {
    background-image: linear-gradient(to top, HotPink 50%, SkyBlue 50%);
}

Por supuesto podemos hacer degradados con múltiples colores simplemente agregando más al final de la lista.


div {
    background-image: linear-gradient(to top, HotPink, SkyBlue, lime);
}

Para conseguir el efecto de bordes planos con más de dos colores es necesario repetir los colores intermedios, o sea, que delimitan con más de un color.


div {
    background-image: linear-gradient(to top, LightSkyBlue 30%, white 30%,
    white 70%, LightSkyBlue 70%);
}

Degradado radial

La sintáxis básica para conseguir un degradado radial es muy similar a la del degradado linear pero, en lugar de linear-gradient, utilizaremos: radial-gradient.


div {
    background-image: radial-gradient(HotPink, RoyalBlue);
}

Podemos forzar que el degradado sea un círculo en lugar de que adopte las proporciones del contenedor con el keyword circle.


div {
    background-image: radial-gradient(circle, HotPink, RoyalBlue);
}

Si quisieramos que el cículo quedase contenido en su totalidad dentro del elemento usaríamos circle closest-side.

Otros posibles valores son closest-corner, farthest-corner, farthest-side.


div {
    background-image: radial-gradient( circle closest-side, HotPink,
    RoyalBlue);
}

Podemos posicionar el gradiente a partir de las coordenadas de su centro, tanto usando unidades como palabras clave.


div {
    background-image:
        radial-gradient(
            circle at top right,
            HotPink,
            RoyalBlue
        );
}
                    

div {
    background-image:
        radial-gradient(
            circle at 20% 60%,
            HotPink,
            RoyalBlue
        );
}

Repeating background

Las propiedades repeating-linear-gradient y repeating-radial-gradient nos permiten repetir el degadado como si de un patrón se tratara. El tamaño del degradado lo definirá el último color-stop.


div {
    background-image:
        repeating-linear-gradient(
            HotPink,
            SkyBlue 30px
        );
}
                    

div {
    background-image:
        repeating-radial-gradient(
            circle,
            HotPink,
            HotPink 20px,
            SkyBlue 20px,
            SkyBlue 50px
        );
}

Combinando degradados

Podemos utilizar múltiples degradados de la misma manera que utilizábamos múltiples imágenes, separándolos con una coma dentro de la propiedad background-image.

Para que los degradados de debajo se vean tendremos que utilizar colores con transparencia.


div {
    background-image:
        radial-gradient(
            circle closest-side, HotPink, rgba(65, 105, 225, .2)
        ),
        linear-gradient(
            orange,olive
        );
}
                    

div {
    background-image:
        radial-gradient(
            circle,
            gold, gold 35px,
            rgba(0,0,0,0) 35px, rgba(0,0,0,0)
        ),
        linear-gradient(
            to top,
            LightSkyBlue 33%,
            white 33%, white 66%,
            LightSkyBlue 66%
        );
}

Las posibilidades son muchísimas y seguirán aumentando cuando los blend-modes propuestos por Adobe tengan soporte completo en todos los navegadores.

Position

Además de los floats, los márgenes y otras propiedades de layout, CSS proporciona la posibilidad de posicionar elementos de manera precisa mediante coordenadas.

Posicionaremos un elemento a partir de de su esquina superior izquierda, de manera muy similar a como lo hacíamos con las imágenes de fondo, utilizando las propiedades top, right, bottom y left (TRBL).

La dificultad consiste en determinar cual será el punto de orígen (0,0) de esas cordenadas y allí es donde entra la propiedad position.

Position static

La propiedad position admite cuatro valores: static, relative, absolute y fixed.

static es el valor por defecto y simplemente ignora cualquier valor de las propiedades TRBL, haciendo que el elemento se comporte de acuerdo a su flow natural o al indicado por la propiedad display.

Position static


.box-1,
.box-2,
.box-3 {
    position: static;
}
.box-1
.box-2
.box-3

Position relative

postion:relative nos permitirá desplazar el elemento de su sitio utilizando como punto 0,0 su esquina superior izquierda natural.

El resto de los elementos de la página no se verán afectados y se comportarán como si el elemento desplazado permaneciera en su sitio.

Position relative


.box-2 {
    position: relative;
        top: 20px;
        left: 40px;
}
.box-1
.box-2
.box-3

Position relative

También podremos usar valores negativos


.box-2 {
    position: relative;
        top: -20px; /* = bottom: 20px */
        left: -40px; /* = right: 40px */
}
.box-1
.box-2
.box-3

Position absolute

position:absolute posicionará al elemento tomando como coordenadas 0,0 a la esquina superior derecha del elemento contenedor, pero siempre y cuando este elemento tenga una propiedad position diferente a static

Si el elemento padre tuviera una posición static, buscará como referencia el siguiente elemento contenedor y asi sucesivamente hasta llegar al body.

Además absolute quitará al elemento de su flow natural por lo que el resto de los elementos actuarán como si no existiese.

Position absolute


.box-2 {
    position: absolute;
        bottom: 0;
        right: 0;
}
.box-1
.box-2
.box-3

Position absolute


.container-1 { position: relative; }
.container-2 { position: static; }
.box-2 {
    position: absolute;
        bottom: 0;
        right: 0;
}
.box-1
.box-2
.box-3

Position absolute

A pesar de la versatilidad que nos proporciona position:absolute, es mejor dejar su uso relegado a pequeños elementos más que a herramienta de layout, ya que si la utilizaramos en todos los elementos el contenido podría ser inaccesible.

Position absolute


.box-1 { position: absolute; bottom: 0; left: 0; }
.box-2 { position: absolute; bottom: 0; right: 0; }
.box-3 { position: absolute; top: 0; right: 0; }
.box-1
.box-2
.box-3

Position absolute


.container-1 { overflow: hidden; }
.box-1 { position: absolute; bottom: 0; left: 0; }
.box-2 { position: absolute; bottom: 0; right: 0; }
.box-3 { position: absolute; top: 0; right: 0; }
.box-1
.box-2
.box-3

Position fixed

position:fixed funcionará de manera similar a position:absolute pero utilizará como referencia para sus coordenadas la ventana del navegador, manteniendo la posición independientemente del scroll de la página.
