Imágenes de fondo

CSS nos permite hacer mucho más con el background de un elemento que simplemente asignarle un color plano.

Gracias a la propiedad background-image podremos asignar imágenes de fondo, pero eso es solo el comienzo.

  • background-color - <color value>
  • background-image - <url (location of image)> | none
  • background-repeat - repeat | repeat-x | repeat-y | no-repeat
  • background-position - <length measurement> | <percentage> | left | center | right | top | bottom
  • background-attachment - scroll | fixed | local
  • background-clip - border-box | padding-box | content-box
  • background-origin - border-box | padding-box | content-box
  • background-size - <length> | <percentage> | auto | cover | contain

* Nuevos en CSS3

Propiedad background

Como sucede con otras propiedades, podemos utilizar la propiedad background para agrupar varias propiedades de las antes mencionadas.

Todas las propiedades son opcionales y pueden ir en cualquier orden, excepto los valores de background-position, primero siempre va el horizontal seguido inmediatamente del vertical.

hay que tener en cuenta que si omitimos un valor este se asignará su valor por defecto.


div {
    background: url(day-06/megaman.png) no-repeat center top content-box;
}

Multiples imágenes de fondo

Podemos utilizar más de una imágen de fondo simplemente separando los valores con una coma.

Si asignamos más de una propiedad relativa al backgrouns tenemos que cuidar que el orden sea el mismo para todas las propiedades.

Si algunas de las imágenes se superponen será la primera de la lista la que quede al frente y el resto le seguirán en orden.


div {
    background:
        url(day-06/dalek.png) 140px top,
        url(day-06/megaman.png) 90px 100px;

    background-repeat: no-repeat;
}

Degradado linear

CSS nos facilita varias opciones para poder crear todo tipo de degradados, una vez más utilizando la propiedad background-image

Un ejemplo de la sintáxis para crear un degrado linear simple sería:

background-image: linear-gradient(180deg, HotPink, SkyBlue)


div {
    background-image: linear-gradient(180deg, HotPink, SkyBlue)
}
